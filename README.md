### 关键特性

* Cortex-A7 960 MHz
* ARM926EJS 202 MHz
* 8051 Core Selectable 32KHz or 202MHz operating speed
* Two IEEE 802.3 10/100M ports with RMII interface
* HDMI TX 1.4, resolution up to 720P
